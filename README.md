# Dogs of the Dow — Stock Screener

A stock screener built with **Next.js** that analyzes the **Dow Jones Industrial Average (DJIA)** specifically using the **Dogs of the Dow** investing strategy.

Inspired by my favourite elective of 1A, Intro to Financial Markets that taught how to pick good stocks! Something we looked out was passive investing strategies and this specific one interested me.

Inspired by Finviz, the project focuses on clean UI, and real-time market data.

## Features 
- DJIA stock screener (30 stocks)
- Preset strategies:
  - Biggest Movers 
  - Top Gainers 
  - Top Losers
  - **Dogs of the Dow** (top 10 dividend yields in DJIA)
  - Live quotes via Finnhub API which auto-refresh every 15 seconds 
  - Dark mode UI with Tailwind CSS

## Tech Stack

- **Next.js (App Router)** + TypeScript
- **Tailwind CSS**
- **Finnhub API** (market data)
